In order to run the program do the following steps:
1. do make in the MSMT directory.
2. read and fill the config.txt file.
3. run using : dflow_calc <path to config.txt file>
NOTE: you can run the program only with analyzed traces, if you wish to run another file that's not in the attached traces please use "script-instruction-set.py" in "Pre-Processing - Translating the traces" to translate yours.
