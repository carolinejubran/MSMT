In order to run the program do the following steps:
1. do make in the Scheduling Algorithms Implementation directory.
2. read and fill the Scheduling Algorithms Implementation/config.txt file.
3. run using : dflow_calc <path to config.txt file>
NOTE: you can run the program only with interpreted traces, run "interpreter.py" in "Pre-Processing - Translating the traces" to translate yours.
